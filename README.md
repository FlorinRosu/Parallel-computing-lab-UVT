# Parallel-computing-lab-UVT

Have installed Visual Studio Code and docker

Open Visual Studion Code on the selected folder where you clone the project:
*code .*

First time you open, it will open (and create) local devcontainer. It is recommended to have the pluggins for cmake and run/debug for C/C++. In the terminal run the following commnads:
```
*mkdir build*
*cd build*
*cmake ../*
*make*
```

And the executable should be created and run: *./hello_openmp*
